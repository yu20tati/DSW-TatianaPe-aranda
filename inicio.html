Objetivo general

Diseñar un OVI con la estructura basica de html5, conociendo cada uno de los pasos para facilitar la creacion de un sitio web.


Objetivos especificos

  • identificar las distintas etiquetas que se usan.
  • conocer los diferentes atributos, que nos permitiran configurar los elementos
  • distinguir los valores que tiene cada atributo.

Concepto html 5

Definicion:

Es un estandar que sirve para definir estructura y el contenido de una pagina web.
Es la quinta version del html y con el los navegadores como Firefox, Chrome, Explorer, Safari y más pueden saber cómo mostrar una determinada página web,
saber dónde están los elementos, dónde poner las imágenes, dónde ubicar el texto


Que son las etiquetas

Las etiquetas son la herramienta fundamental para que los navegadores puedan interpretar el código y permitirnos ver imágenes, texto, párrafo, y estructuras.
Las etiquetas a las que estábamos acostumbrados, por otro lado, introducen un nuevo funcionamiento. El caso ejemplo es el de las etiquetas header y footer, que, como dijimos,
ahora permiten separar las secciones, y no solamente el comienzo y el fin de una página.


Que son los atributos

son la forma que tienen los autores de definir propiedades para un elemento. Estas propiedades habitualmente cambian la forma en que los navegadores interpretan al elemento,
al cambiar su significado o presentación. Por ejemplo, el elemento a inserta un vínculo en el documento, pero el atributo rel indica la relación entre el documento actual
